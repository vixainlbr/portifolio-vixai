Task Management System

Sistema de Gestão de Tarefas

Overview

Visão Geral

Fullstack web application for creating, tracking and managing tasks across teams. Features status updates, deadlines, and user comments.

Aplicação web fullstack para criação, acompanhamento e gestão de tarefas entre equipes. Inclui atualizações de status, prazos e comentários.

Technologies Used

Tecnologias Utilizadas

  • FastAPI
  • PostgreSQL
  • React + Tailwind CSS
  • JWT Authentication
  • REST API + Fetch

Problem Solved

Problema Resolvido

Many teams struggle to keep tasks visible, structured and synchronized. This system brings clear visibility, filtering and real-time status updates.

Muitas equipes têm dificuldade em manter tarefas visíveis, estruturadas e sincronizadas. Este sistema traz visibilidade clara, filtros e atualizações em tempo real.
