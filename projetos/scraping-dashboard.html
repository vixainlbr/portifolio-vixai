Data Scraping Bot with Dashboard

Bot de Extração de Dados com Dashboard

Overview

Visão Geral

System that scrapes public data from websites and presents it in an interactive dashboard. The tool is ideal for market monitoring, competitive analysis, and public data tracking—eliminating the need for manual updates and centralizing insights.

Sistema que coleta dados públicos de sites e os apresenta em um dashboard interativo. A ferramenta é ideal para monitoramento de mercado, análise da concorrência e acompanhamento de dados públicos—eliminando atualizações manuais e centralizando os insights.

Technologies Used

Tecnologias Utilizadas

  • Python
  • BeautifulSoup + Selenium (web scraping)
  • Streamlit (interactive dashboard)
  • Pandas (data cleaning and processing)
  • Agendadores e scripts automatizados

Problem Solved

Problema Resolvido

Manually checking sites for updates is inefficient and error-prone. This bot automates the extraction process, transforming raw HTML into useful insights presented in real time.

Verificar sites manualmente é ineficiente e propenso a erros. Este bot automatiza a extração, transformando HTML bruto em insights úteis apresentados em tempo real.
