PDF Route Indexer

Indexador de Rotas a partir de PDF

Overview

Visão Geral

Python application that extracts route codes from logistics PDF manifests and generates a clean, formatted PDF index for fast assignment.

Aplicativo em Python que extrai códigos de rotas de manifestos logísticos em PDF e gera um índice em PDF formatado para facilitar a distribuição.

Technologies Used

Tecnologias Utilizadas

  • Python
  • PyMuPDF (fitz)
  • FPDF
  • Tkinter
  • Pandas

Problem Solved

Problema Resolvido

Tired of sorting through 50, 80, even 100-page delivery manifests?
All you really need is a clean summary: routes, codes, and what’s still missing.
This Python-based system automates the process — it reads the original PDF, extracts only the key data, and generates a single formatted report.
The output is fully customizable by team, route, or delivery type.
Less paper. Less ink. Less wasted time.
In real use, companies have cut print volume by over 90% and sped up the distribution process. This is automation that actually fits your logistics workflow.

Cansado de lidar com manifestos de 50, 80 ou até 100 páginas?
Você só precisa de uma visão objetiva: rotas, códigos e o que ainda falta entregar.
Este sistema em Python faz isso de forma automatizada — lê o PDF original, extrai apenas as informações essenciais, formata o conteúdo em um único documento final e ainda permite personalizações por equipe, por rota ou por tipo de entrega.
Menos papel, menos tinta, menos tempo perdido.
Em testes reais, empresas reduziram em mais de 90% o volume de impressão e ganharam agilidade na separação e distribuição das rotas. É automação prática, eficiente e sob medida para a rotina logística.
