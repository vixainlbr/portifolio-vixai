PDF Route Indexer

Indexador de Rotas a partir de PDF

Overview

Visão Geral

Python application that extracts route codes from logistics PDF manifests and generates a clean, formatted PDF index for fast assignment.

Aplicativo em Python que extrai códigos de rotas de manifestos logísticos em PDF e gera um índice em PDF formatado para facilitar a distribuição.

Technologies Used

Tecnologias Utilizadas

  • Python
  • PyMuPDF (fitz)
  • FPDF
  • Tkinter
  • Pandas

Problem Solved

Problema Resolvido

Manual route indexing from manifests is slow and error-prone. This tool automates the process, ensuring consistent formatting and saving time for logistic operators.

O processo manual de indexação de rotas em manifestos é lento e sujeito a erros. Esta ferramenta automatiza esse trabalho, padronizando o layout e economizando tempo dos operadores.
