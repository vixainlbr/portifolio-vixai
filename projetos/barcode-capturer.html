Barcode Capturer Script

Script Capturador de Códigos de Barras

Overview

Visão Geral

A Python-based script that captures barcode input via keyboard scanners and logs the data into a CSV file with timestamps.

Um script em Python que captura entradas de códigos de barras via leitores conectados como teclado e salva os dados em um arquivo CSV com data e hora.

Technologies Used

Tecnologias Utilizadas

  • Python
  • keyboard
  • pandas
  • datetime, os
  • CSV file handling

Problem Solved

Problema Resolvido

Operators often need to quickly register scanned codes from packages or assets. This script automates logging of each scanned value with its timestamp, avoiding the need for manual input.

Operadores muitas vezes precisam registrar rapidamente códigos escaneados de pacotes ou ativos. Este script automatiza o registro de cada valor com data e hora, evitando entradas manuais.
