Public Data Scraping Dashboard

Dashboard de Extração de Dados Públicos

Overview

Visão Geral

System that scrapes public data from selected websites, processes the information, and displays results in an interactive, user-friendly dashboard. Ideal for market tracking, competitor monitoring, or regulatory updates.

Sistema que extrai dados públicos de sites específicos, processa as informações e exibe os resultados em um painel interativo e intuitivo. Ideal para monitoramento de mercado, concorrência ou atualizações regulatórias.

Technologies Used

Tecnologias Utilizadas

  • Python (BeautifulSoup / Selenium)
  • Pandas para tratamento de dados
  • Streamlit para criação de dashboard
  • Requests / APIs públicas
  • Deploy via Streamlit Cloud

Problem Solved

Problema Resolvido

Manual tracking of market or regulatory data is time-consuming and prone to errors. This solution automates collection and visualization, giving teams real-time access to structured public data without coding.

O monitoramento manual de dados de mercado ou regulatórios consome tempo e é sujeito a falhas. Esta solução automatiza a coleta e visualização, oferecendo acesso em tempo real a dados públicos estruturados sem necessidade de programação.
